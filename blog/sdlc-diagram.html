1: Issue & Spec 👤 Human

Architect syncs Jira story to a clean GitHub Issue (SPEC.md).

Hygiene: Define anti-goals. Do not write pseudo-code.
2: Implementation 🤖 AI

AI reads SPEC.md, forks branch, writes code, runs local tests.

Hygiene: Monitor AI token limits. Restrict PR sizes.
3: Quality Gates
⚙️ Compute 🤖 AI

GitHub Actions runs tests. AI Reviewer checks logic.

Hygiene: Goodhart's Law. Exhaustive unhappy-path testing.
4: Review & Collab
👤 Human 🤖 AI

Maintainer tests UX, and collaborates with AI via PR comments.

Hygiene: Verify logic. Reject large spaghetti PRs.
5: Merge
👤 Human ⚙️ Compute

Maintainer approves. Git mechanisms safely merge code.

Hygiene: Enforce strict linters/formatters upon merge.
6: Deploy ⚙️ Compute

CI/CD automation packages and pushes deterministic build.

Hygiene: Chaos Engineering. Maintain DR backups.
👤 HUMAN (Architect / Maintainer)
🤖 AI (Implementation / Review)
⚙️ COMPUTE (Deterministic CI/CD)
The Architect-Maintainer defines specifications and oversees the AI Implementation, while Deterministic Automation enforces rigorous quality gates. The human remains the ultimate gatekeeper before production deployment.
